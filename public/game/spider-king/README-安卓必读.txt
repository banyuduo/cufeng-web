蛛王传说 — 在安卓手机上如何打开（离线包）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
一、若屏幕上是一大段「代码」
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

若你看到的是 <!DOCTYPE html>、<script>、很多英文等，说明手机把网页当成
「文本 / 代码」打开了，并没有用浏览器去运行游戏。

请按下面「正确打开方式」操作，不要用「记事本」「HTML 查看器」「代码预览」
等应用打开。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
二、正确打开方式（推荐）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 先把压缩包完整解压到手机存储里的某个文件夹（不要只在聊天软件里点压缩包
   里的「预览」）。

2. 在文件夹里找到并记住：游戏入口文件名叫  index.html
   （不要打开 sw.js、manifest.webmanifest 等，那些不是游戏界面。）

3. 安装或使用 Chrome 浏览器（谷歌浏览器）。

4. 用系统「文件管理」进入解压后的文件夹，长按 index.html：
   选「打开方式」或「用其他应用打开」→ 选择「Chrome」或「浏览器」。

   若列表里没有 Chrome：先到应用商店安装 Chrome，再重复第 4 步。

5. 打开后应看到游戏标题「蛛王传说」和画面，而不是满屏代码。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
三、仍异常时
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

· 确认解压后的文件夹里仍有 sprites 子文件夹（勿删、勿只拷贝单个 html）。
· 可尝试：先打开 Chrome，在地址栏输入 file:// 再按提示从存储里选文件
  （部分机型支持从菜单里选本地文件）。
· 部分品牌「内置浏览器」对本地文件支持不好，优先用 Chrome。
