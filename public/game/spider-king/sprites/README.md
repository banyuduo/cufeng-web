# 蛛王传说 - 精灵图说明

本目录存放游戏角色与敌人的图片素材。替换以下文件即可更改游戏形象：

| 文件名 | 用途 | 建议尺寸 |
|--------|------|----------|
| `player.png` | 玩家角色（骑士） | 40×60 或等比例 |
| `boss.png` | 蛛王 BOSS | 130×105 或等比例 |
| `minion.png` | 小蜘蛛敌人 | 26×22 或等比例 |

- 任意图片缺失时，游戏会自动回退到程序绘制形象
- 当前素材来源：Kenney（CC0）、OpenGameArt / bevouliin.com（CC0）
- 可替换为您自己的设计图或从 [OpenGameArt](https://opengameart.org)、[Kenney](https://kenney.nl) 等下载免费素材
